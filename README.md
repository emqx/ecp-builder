# ECP builder

This repository contains the ECP builder used for building ECP package and container images.


## Requirements

1. ECP backend & frontend repositories must have the same tag.
2. Create a same tag to this repository to trigger the build.


## package & container image

1. saved as `ghcr.io/emqxecp/[container-image-name]:<tag>`
2. available at [GitHub Container Registry](https://github.com/orgs/emqxecp/packages)
3. install package available at releases [GitHub Releases](https://github.com/emqx/ecp-builder/releases)
